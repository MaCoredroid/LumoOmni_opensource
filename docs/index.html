Open Research Blueprint

Discrete multimodal tokenization, staged for reliability and scale.

LumoOmni is a staged research program for building an any-to-any multimodal foundation model with discrete tokens. It starts with a Qwen3‑8B + SigLIP VLM baseline (resampler/projector), advances to a Unified Token Interface built on SEED + EnCodec, and formalizes tokenized WebDataset pipelines with token-space hashing and deterministic audits. This site organizes the specs, verification gates, and artifacts so researchers can reproduce the system and HR teams can understand scope, rigor, and outcomes.

Program Snapshot

  • Stage 1 VLM baseline: Qwen3‑8B + SigLIP + resampler/projector
  • Stage 1.1 eval hardening: token‑weighted buckets + golden set
  • Stage 2 tokenized WebDataset shards with token_space hashing
  • Stage 3 audits: UTI determinism, dataset verification, decode gates
