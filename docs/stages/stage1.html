Stage 1

Alignment pretraining (captioning-style)

Build the VLM baseline by training only the connector while keeping the LLM and vision tower frozen.

Goals

Teach the LLM to condition on injected visual tokens without degrading text behavior.

Deliverables

Connector training loop, data schema, prompt template, and deterministic evaluation.

Verification

Teacher-forced ablations, truncation checks, and label coverage metrics.
